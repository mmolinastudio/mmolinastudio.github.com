Instalar MongoDB en Ubuntu 13.04

2013-05-08 databases, mongodb, linux, ubuntu

El mes pasado empecé un proyecto usando MySQL. En principio iba a ser una web para uso local pero poco a poco empecé a añadirle cosas, a pensar en algo más grande y ahora estoy viendo que si quiero hacerlo bien, necesito una Base de Datos más eficiente. Nunca he usado NoSQL y creo que este es un buen momento para empezar.

¿Y por qué decantarme por MongoDB y no por CouchDB, Cassandra, BigTable o cualquier otra? Simplemente porque estoy usando la librería IonAuth de Codeigniter, que usa InnoDB o MongoDB, y no quiero complicarme la vida. La migración será mucho más facil.

Os dejo una pequeña intrducción de qué es MongoDB (by Wikipedia):

MongoDB (de la palabra en inglés “humongous” que significa enorme) es un sistema de base de datos NoSQL orientado a documentos, desarrollado bajo el concepto de código abierto.

MongoDB forma parte de la nueva familia de sistemas de base de datos NoSQL. En vez de guardar los datos en tablas como se hace en las base de datos relacionales, MongoDB guarda estructuras de datos en documentos tipo JSON con un esquema dinámico (MongoDB llama ese formato BSON), haciendo que la integración de los datos en ciertas aplicaciones sea más fácil y rápida.

El desarrollo de MongoDB empezó en octubre de 2007 por la compañía de software [10gen](http://www.10gen.com/ 10gen). Ahora MongoDB es una base de datos lista para la producción de uso y con muchas características (feature). MTV Network, Craiglist y Foursquare4 son algunas de las empresas que utilizan esta base de datos.

Instalación

Puesto que Ubuntu no provee los paquetes mongodb, mongodb-server, y mongodb-clients por defecto, hemos de añadir antes el repositorio de 10Gen al Sistema de Gestión de Paquetes de Ubuntu (Centro de Software), importando antes la clave GPG que asegura la autenticidad de los paquetes de 10Gen, y finalmente instalar la última versión estable de MongoDB. Tan sencillo como esto:

$ sudo apt-key adv --keyserver keyserver.ubuntu.com --recv 7F0CEB10
$ echo 'deb http://downloads-distro.mongodb.org/repo/ubuntu-upstart dist 10gen' | sudo tee /etc/apt/sources.list.d/10gen.list
$ sudo apt-get update
$ sudo apt-get install mongodb-10gen

Si queremos que se ejecute al inicio del sistema: $ sudo update-rc.d mongodb defaults

Control

Para iniciar, parar o reiniciar MongoDB: $ sudo service mongodb start $ sudo service mongodb stop $ sudo service mongodb restart Podemos comprobar que se ha iniciado correctamente entrando en http://localhost:28017 o leyendo el archivo /var/log/mongodb/mongodb.log

Uso

Con el comando $ mongo se nos abrirá el shell de MongoDB. Podemos hacer un sencillo test para ver si se está ejecutando correctamente:

> db.test.save( { a: 1 } )
> db.test.find()

Interfaces gráficas

Aunque aún me queda probarlas, he hecho una selección de las GUIs que más me han gustado.

  • phpMoAdmin. Lo básico para consultar y editar BDs, importar/exportar ficheros json,... en un solo archivo php de 95kb.
  • Rockmongo. También escrita en php pero con más opciones que la anterior. (e.g. creación de BDs, gestión de usuarios,...)
  • GenghisApp y Mongo3. Opciones Ruby ineresantes.
  • Fang of Mongo. Opción Python/Django.
  • RoboMongo. En inglés se define como "shell-centric cross-platform open source MongoDB management tool". Traducir shell-centric es imposible. Si alguien lo sabe explicar, por favor que deje un comentario. Yo supongo que se refiere a que el programa está basado en el shell de mongo, y simplemente ofrece una ayuda gráfica para escribir los comandos. Es posible que me equivoque y sea más potente, pero hasta que no lo pruebe no puedo hablar. Lo que sí está claro es que RoboMongo es una Aplicación multiplataforma escrita en C++ y sigue en pleno desarrollo.
  • UMongo. De momento quiero una interfaz web y UMongo es una aplicación Java, pero la dejo en la lista por estar respaldada por la comunidad, tener buenas opiniones, y como las anteriores, ser multiplataforma y open source.

Instalar phpMoAdmin y RockMongo

Para instalar cualquiera de estas interfaces, antes hemos de instalar el driver de php para mongo de esta forma:

$ sudo apt-get install php5-dev php-pear
$ sudo pecl install mongo

y añadir al archivo php.ini (ubicado en /etc/php5/apache2/) la línea extension=mongo.so.

La instalación es tan sencilla como descargar y descomprimir ambas interfaces en la carpeta /home/username/www/ o donde tengas tus archivos públicos de Apache, y acceder a http://localhost/moadmin.php o http://localhost/rockmongo (username:admin, password:admin). Recuerda reiniciar antes los servicios de apache y mongodb.
